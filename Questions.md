# Essais sur une nomenclature d'idéologies opposées

## Définition de paires d'idéologies opposées avec les questions attribuées

Paires d'idéologies antagonistes classées par thèmes. Les questions sont sujètes à modification et les proposition sont les bienvenues. N'hésitez pas à poster dans la section "Issues" du github ou même à proposer des pull requests.
