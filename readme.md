# About

Ce site est une proposition de test d'orientation politique. Il est inspiré de politiscales.
Il fonctionne avec un simple script javascript, et du html/css.

C'est principalement un projet pour m'entraîner en JS.

## Stade de développement

Le projet est en cours de développement.
Le test est déjà fonctionnel, des questions sont disponibles mais elles ne sont pas définitives. Les catégories ne sont pas non plus définitives.
